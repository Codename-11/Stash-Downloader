Features

🎥

Video Downloads

Download videos from hundreds of sites using yt-dlp. Supports quality selection and format preferences.

🖼

Image & Gallery Support

Download images and galleries from booru sites like Rule34, Gelbooru, and Danbooru with full tag extraction.

🏷

Automatic Metadata

Extracts titles, descriptions, tags, and performers automatically. Maps to your existing Stash library.

📂

Queue Management

Queue multiple downloads, track progress, retry failed items. Queue persists across page refreshes.

🔌

Browser Extension

Firefox extension to send URLs directly to your download queue from any page with one click.

🔒

Proxy Support

Configure HTTP/SOCKS proxy to bypass geo-restrictions and access region-locked content.

Installation

1

Add Plugin Source

Go to Settings → Plugins → Available Plugins and click "Add Source"

https://codename-11.github.io/Stash-Downloader/index.yml
2

Install Plugin

Find "Stash Downloader" in the list and click Install

3

Install yt-dlp

The plugin requires yt-dlp for video downloads. Install via pip:

pip install yt-dlp
4

Access Downloader

Click the download icon in Stash's navbar or navigate to /plugin/stash-downloader

1

Download Extension

Get the latest stash-downloader-extension.zip from GitHub Releases

2

Install in Firefox

Navigate to about:addons, click the gear icon (⚙), and select "Install Add-on From File..."

Select the downloaded ZIP file to install.

3

Configure Connection

Click the extension icon in your toolbar, then click the gear icon (⚙) to open settings:

  • Stash URL: Your Stash server address (e.g., http://localhost:9999)
  • API Key: Required if Stash authentication is enabled (Settings → Security)
4

Start Sending URLs

You're ready! Use the extension in three ways:

  • Right-click a link → "Send to Stash Downloader" → Choose content type
  • Highlight a URL → Right-click → "Send to Stash Downloader"
  • Click extension icon → Send current page URL via popup
✓

Verify Connection

The extension icon shows connection status: green = connected, red = disconnected. URLs appear in your Stash Downloader queue instantly when connected!
