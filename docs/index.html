Installation

1

Add Plugin Source

Go to Settings → Plugins → Available Plugins and click "Add Source"

https://codename-11.github.io/Stash-Downloader/index.yml
2

Install Plugins

Find Stash Downloader and/or Stash Browser in the list and click Install

3

Install yt-dlp (for Downloader)

Stash Downloader requires yt-dlp for video downloads. Install via pip:

pip install yt-dlp
4

Access Plugins

Click the icons in Stash's navbar or navigate to:

  • /plugin/stash-downloader - Stash Downloader
  • /plugin/stash-browser - Stash Browser
